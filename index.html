Adoção de Cachorros

Este foi um projeto de desenvolvimento de um site de adoção de cachorros, criado utilizando o Bootstrap. O objetivo do site era facilitar a adoção responsável, apresentando informações sobre os cães disponíveis para adoção de forma simples e direta. O projeto permitiu aplicar conceitos de design responsivo e usabilidade, além de aprimorar habilidades no uso do Bootstrap para criar um layout funcional e acessível.
